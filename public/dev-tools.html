🧹 清除 localStorage

清除瀏覽器暫存資料（不影響資料庫）

說明：清除 localStorage 後需重新登入，但資料庫資料不受影響。

💣 刪除資料庫使用者

從資料庫完全刪除當前使用者資料（包含所有評分、MBTI、徽章等）

⚠️ 危險操作：刪除後無法復原！所有遊戲評分、MBTI、成就都會消失。

🔄 完整重置（新使用者模式）

清除所有本地資料 + 刪除資料庫使用者 = 完整回到新使用者狀態

⚠️ 最危險操作：這會：
1. 刪除資料庫中所有相同 Google ID 的帳號（包含舊重複帳號）
2. 清除所有 localStorage
3. 重新載入頁面

下次登入會視為全新使用者！

🧹 清除重複帳號（不重置登入）

找出資料庫中所有相同 Google ID 的帳號並全部刪除，但保留本地登入狀態。
用於：登入後仍繼承舊帳號資料（如「幽默的兔子 #8143」）時使用。


                

            


            

                📋 操作步驟：

                1. 點「掃描重複帳號」確認資料庫狀態

                2. 點「刪除所有我的 DB 帳號」清除全部

                3. 留在頁面（不要登出）

                4. 去「我的檔案」重新設定 MBTI → 會建立乾淨的新帳號

                5. 再去「桌遊探索」重新評分即可
